# ProwlrBot Blog

> Humanized posts about what we're building, why it matters, and where we're headed.

## Categories

| Tag | Purpose |
|-----|---------|
| `launch` | New features, releases, milestones |
| `guide` | Tutorials, how-tos, walkthroughs |
| `update` | Progress reports, what changed this week |
| `deep-dive` | Architecture decisions, technical breakdowns |
| `alert` | Security advisories, breaking changes, urgent fixes |
| `vision` | Where ProwlrBot is going, philosophy, roadmap |

## Writing Style

- **First person plural** — "we built", "we're shipping"
- **Conversational** — like talking to a smart friend, not a corporate blog
- **Show don't tell** — code snippets, screenshots, terminal output
- **Honest** — if something's rough, say so. People respect that.
- **Short paragraphs** — 2-3 sentences max per paragraph

## File Naming

```
YYYY-MM-DD-slug-title.md
```

Example: `2026-03-10-war-room-is-live.md`
