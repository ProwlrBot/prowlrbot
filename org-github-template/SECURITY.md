# Security Policy

## Reporting Vulnerabilities

For security vulnerabilities in any ProwlrBot repository, please follow the responsible disclosure process in our main repository:

**[ProwlrBot Security Policy](https://github.com/prowlrbot/prowlrbot/blob/main/SECURITY.md)**

Do NOT open public issues for security vulnerabilities. Use GitHub's private vulnerability reporting feature or email the maintainers directly.

## Scope

This policy covers all repositories under the `prowlrbot` organization.
