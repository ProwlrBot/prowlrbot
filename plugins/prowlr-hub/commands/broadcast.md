# Broadcast

Send a status update or message to all connected agents in the war room.

## Instructions

If the user provides arguments, use them as the message. Otherwise, ask what to broadcast.

1. Call the `broadcast_status` MCP tool with the message
2. Confirm the broadcast was sent
3. Show how many agents will see it

## Examples

- `/broadcast Hit a blocker on the auth middleware — need help`
- `/broadcast API endpoints are done, ready for frontend integration`
- `/broadcast Taking a break, back in 30 min`
